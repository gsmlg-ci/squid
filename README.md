# squid

[![Build](https://github.com/gsmlg-ci/squid/actions/workflows/build.yml/badge.svg)](https://github.com/gsmlg-ci/squid/actions/workflows/build.yml)

Docker image published to:
- `docker.io/gsmlg/squid`
- `ghcr.io/gsmlg-dev/squid`

## Usage

```bash
# From Docker Hub
docker pull gsmlg/squid:latest

# From GitHub Container Registry
docker pull ghcr.io/gsmlg-dev/squid:latest
```

## Build

```bash
docker build -t gsmlg/squid .
```
